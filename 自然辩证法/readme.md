# 我感觉我似乎是走火入魔了。

理解到导师所言，”讲故事“的重要性。
科研的本质是讲故事。

讲故事
客观实际上是客观的主观，主观实际上是主观的主观。
看似是在研究真实世界，实际上还是在研究人的思想。
无论是同行评议制度还是假说演绎，自我论证，消融实验，本质上是让个人的主观臆断，变成个人的客观的主观臆断，再变成一种群体的主观臆断，再是群体的客观的主观而已。
关于这个主观臆断是否真的符合客观事实，主不在乎。
我似乎抓到了问题的关键。

实用主义：通过实用价值让别人相信这个技术内涵的科学原理。所述科学原理是一种群体性的主观判断，可能具有一定程度上符合真实世界运行规律的性质。

从这个角度上讲，科研所做的事情和古老时代讲故事的巫医并没有本质区别。

# 什么是自由

马经的反例：没有压迫就没有自由。

* 自由本身就是由压迫定义的。

* 人对自由的追求是无止境的：蹲着想要站着，站着想要坐着，坐着还想要躺着。

* 马经说资本异化了人。那么如果没有异化，那么就异化了异化本身。这是一个非常有趣的事情。

  

  ## 被限制反而是自由。

笼中鸟在笼子里想要飞出去，飞出去就是自由。此时的自由的具象化的，可以得到的。

然而，飞出去之后，此时的自由又是什么呢？此时的自由的难以实现的，难以具象化的。反而没有了自由。
